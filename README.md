# Ethereal Tower RPG (虛空之塔 RPG)

**Ethereal Tower RPG** 是一款基於網頁的**現代儀表板風格 (Dashboard UI)** 文字冒險角色扮演遊戲。 v3.8.0 版本加入了全新的**打工系統**與**摸魚模式**，讓冒險與生活取得平衡。

打破傳統文字遊戲需要頻繁捲動的缺點，本遊戲採用**三欄式單頁設計**，讓玩家能一眼掌握角色狀態、戰鬥過程與背包資訊。遊戲結合了嚴謹的數值養成系統、經濟體系與永久死亡機制。

## 🌟 核心特色

### 1. 🖥️ 現代化儀表板介面 (Modern Dashboard UI)
*   **一目了然**：全新三欄式佈局，左側為角色與技能，中間為戰鬥核心，右側為地圖與商店。
*   **全域伺服器統計**：即時顯示全伺服器玩家的死亡數、倖存者與最高樓層紀錄。
*   **智能懸浮視窗**：裝備與技能資訊採用全域懸浮層，支援裝備數值比較。
*   **社交分享**：內建分享按鈕，可一鍵複製遊戲連結分享給好友。

### 2. 💼 打工與生活 (Work & Life) - **NEW!**
*   **打工系統**：冒險累了嗎？在城鎮中選擇「臨時工」或「血汗大夜班」來賺取穩定金幣。打工期間探索功能將鎖定，適合掛機賺旅費。
*   **摸魚模式 (Stealth Mode)**：上班上課偷玩？按下 **Alt + S** 或點擊介面上的幽靈圖示，瞬間切換至偽裝的「404 Not Found」畫面，完美隱藏遊戲視窗。
*   **休息冷卻**：回家休息不再是無限制的，需要等待冷卻時間，增加了資源管理的策略性。

### 3. 💰 經濟與補給系統
*   **多元藥劑**：除了生命藥水，現在商店還販售 **力量/敏捷/智力/幸運** 四種屬性藥水，能在單場戰鬥中提供強力 Buff。
*   **雜貨商店**：在城鎮中可開啟商店購買冒險物資。
*   **自動化配置**：玩家可設定「自動喝水」的生命值門檻（例如 HP < 50% 自動使用），讓掛機戰鬥更安全。

### 4. 🛡️ 職業與戰鬥機制
*   **五大職業**：騎士 (Tank)、戰士 (Balance)、盜賊 (Crit/Dot)、弓箭手 (Hit/Dot)、法師 (Skill/Heal)。
*   **狂暴機制**：BOSS 在低血量時會進入**狂暴狀態**，攻擊力與技能頻率大幅提升，並伴隨視覺震動特效。
*   **技能專精**：角色等級提升後自動解鎖技能，技能包含暈眩、中毒、燃燒或屬性提升效果。
*   **自動戰鬥**：提供「開關式」自動戰鬥功能，適合掛機練功。

### 5. ☠️ 永久死亡與挑戰
*   **硬核機制**：角色的生命值 (HP) 一旦歸零，該角色的旅程即刻終結。
*   **BOSS 挑戰與警示**：每 10 層設有強力 BOSS，進入前會有危險警示（可選擇不再顯示以加速挑戰節奏）。
*   **死亡回顧**：保留最後的戰鬥日誌 (Log)，讓玩家能分析死因後再重新開始。

## 🎮 操作指南

1.  **創建角色**：輸入名稱並選擇職業（可輸入邀請碼獲得特殊獎勵）。
2.  **摸魚模式**：按下 **Alt + S** 進入/退出偽裝畫面（或向上滾動滑鼠滾輪退出）。
3.  **戰鬥與掛機**：點擊區域進行探索，開啟 Auto 可自動戰鬥。
4.  **打工**：在城鎮點擊「打工」按鈕，選擇工作後即可放置等待金幣入帳。
5.  **背包管理**：右鍵點擊物品可快速出售，或使用「一鍵出售」清理雜物。
